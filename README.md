# MealMap

MealMap helps students find **free and cheap food on campus**, ranked by what’s worth walking to.

Browse what’s on the pass right now, filter by where you are and when you need food, and see which options are still available. Spotted something? Add it in seconds, paste an event post, or ask MealMap out loud what’s nearby.

Built for UNSW — one place to catch society events, club food, and crowd updates before it’s gone.
